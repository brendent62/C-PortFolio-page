HTTP C socket website - This Website

Designed and implemented a production-grade HTTP web server from scratch in C using low-level POSIX socket programming, handling TCP connections, manual HTTP request parsing, routing, and static file delivery without external frameworks. Deployed the server on a cloud VM using Amazon Web Services EC2 and integrated Cloudflare as a reverse proxy for HTTPS encryption, origin IP protection, and DDoS mitigation. Implemented a full CI/CD pipeline using GitHub and GitHub Actions to automatically build, test, and deploy updates to both staging and live production environments, enabling rapid and reliable iteration. Containerized the application with Docker and configured Linux networking, DNS, and firewall rules to securely expose services while preventing direct origin access. Monitored real-world traffic and exploit attempts and hardened the system by restricting access paths and mitigating automated vulnerability scans.

Tech:
C, POSIX Sockets, Linux, TCP/IP, HTTP, AWS EC2, Cloudflare, Docker, GitHub Actions, CI/CD, Networking, Systems Programming, Security Hardening
